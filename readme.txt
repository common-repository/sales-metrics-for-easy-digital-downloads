Baremetrics style sales metrics report for Easy Digital Downloads!

== Description ==

The Sales Metrics for Easy Digital Downloads is a free add-on developed by the good people at <a href="http://phpcontrols.com">phpcontrols.com</a>. 

The add-on is supported by all major desktop web browsers as well as on mobile and tablet devices.

= Features =

* Total customers
* Total revenue
* Total sales
* Top sellers
* One-click filter by date range
* Recent purchase by date range!

= System Requirements =

* Easy Digital Downloads version 2.x. 

= I've got suggestions. Where should I send them? =

Great! Please email <a href="mailto:info@eddsm.com">info@eddsm.com</a> and submit your ideas and suggestions!

== Installation ==

Please follow these simple steps to install the plugin:

1. From the dashboard of your site, navigate to Plugins --> Add New.
2. Select the Upload option and hit "Choose File."
3. When the popup appears select the easy-digital-downloads-sales-metrics-x.zip file from your desktop. (The 'x' will change depending on the current version number).
4. Follow the on-screen instructions and wait as the upload completes.
5. When it's finished, activate the plugin via the prompt. A message will show confirming activation was successful.

That's it! You can now access "Sales Metrics" link on the nav bar in admin dashboard.
